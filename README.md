# 📝 React Todo App

A fully responsive and modern Todo Application built using React JS with complete CRUD functionality, dynamic rendering using `.map()`, reusable components, and status filtering.

## ✨ Features

- ➕ Add New Todos
- 📖 Display Todos Dynamically using `.map()`
- ✏️ Edit Todo Name & Description
- 🔄 Update Todo Status
- ❌ Delete Todos
- 🔍 Filter Todos by Status
- 📱 Fully Responsive Design
- ♻️ Reusable React Components
- 🎨 Modern Card-Based UI
- 🟢 Status Color Indicators
